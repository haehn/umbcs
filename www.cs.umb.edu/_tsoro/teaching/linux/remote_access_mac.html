Software Needed

  • The Mac comes already has the software you will need to work on a Unix machine
  • The Mac OS is really a customized version of Unix
  • Like all versions of Unix, it has a command line
  • To see the command line, you need to run the Terminal application ...
  • which you will find in the Utilities folder ...
  • inside the Applications folder
  • This will open a window ...
  • where you can talk to your Macintosh through the command line
  • The two programs that come with the Mac are
    • ssh (secure shell)
    • scp (secure copy)
  • To make copying files to the UMB Unix machines easier ...
  • you can use FileZilla instead of scp
  • You can download FileZilla from https://filezilla-project.org

Logging in to a Unix Machine

  • Open the Macintosh Terminal application
  • At the command line enter 
    ssh USERNAME@users3.cs.umb.edu
    where USERNAME is a placeholder ...
  • for your Unix username
  • You will be prompted for your password
  • After you enter it, you get a Unix command line
  • To log into users3 I would type 
    ssh ghoffman@users3.cs.umb.edu

Copying Files to a Unix Machine

  • Run FileZilla
  • In the Host name box, enter users3.cs.umb.edu
  • In the User name box, enter your Unix username
  • In the Password box, enter your Unix password
  • In the Port box, enter 22
  • Click the Quickconnect button
  • You will now see a window with two panes
  • The pane on the left shows the files on your PC
  • The pane on the right show the Unix directory structure
  • You can click and drag a file from your PC to the Unix machine
  • But you can only do this in directories where you have write privileges

Copying Files to a Unix Machine - The Hard Way

  • There is another way to copy files from a Mac to a Unix Machine
  • It involves using the command line
  • Open the Macintosh Terminal application
  • You will find it in the Utilities folder ...
  • inside the Applications folder
  • Using the Unix cd (change directory) command ...
  • go to the directory (folder), that contains the file you want to transfer
  • Then at the command line enter 
    scp  FILENAME  USERNAME@users3.cs.umb.edu:
  • where FILENAME is the name of the file you want to copy ...
  • and USERNAME is your Unix username
  • You will be prompted for your password
  • Be sure to include the the colon, :, after users3.cs.umb.edu
  • If you don't nothing will happen
  • This will transfer the file to your home directory
  • From there you can use the Unix mv (move) command ...
  • to move the file to the proper directory
  • If you wanted to move the file hello.py to the directory ...
  • whose relative pathname is it116/code_entries/ex_01 ...
  • you would type 
    mv hello.py it116/code_entries/ex_01
  • You can save yourself a step ...
  • by copying the file directly to the correct directory
  • You do this by following the colon with the path to the directory 
    scp FILENAME USERNAME@users3.cs.umb.edu:PATH_TO_DIRECTORY
  • If I wanted to copy the file hello.py to my it116/hw/hw1 directory ...
  • I would enter the following at the command line 
    scp hello.py ghoffman@users3.cs.umb.edu:it116/hw/hw1

Accessing a Unix Machine Using the UMB Wireless Network

  • If you are connected to the UMB Wireless network ...
  • you can use ssh, scp and FileZilla ...
  • just as you do from home
