IT 244: Introduction to Linux/Unix
Answers to Class 3 Ungraded Quiz

  1. What command would you use to keep a record of every command you type and the output of those commands? 
    script
  2. What key would you press to get the last command you typed? 
    ↑
  3. What keys would you hit to go to the beginning of the command line? 
    Control A
  4. What keys would you hit to go to the end of the command line? 
    Control E
  5. What is the text editor used by most system administrators? 
    vi or vim
  6. How do you you abort a running program? 
    Control C
  7. What would you type at the command line to get the man page for the ls command? 
    man ls
  8. How do you get the next screenfull when using man? 
    hit the Space bar
  9. How do you exit the man program? 
    hit q
  10. What is the other program, besides man, that you can use to get help? 
    info
