IT 244: Introduction to Linux/Unix
Class 1 Exercise

Purpose

The purpose of this exercise is to create class directory for this course, and, if you don't already have one, to create a Unix account.

It is important that you complete this exercise now, because you will not be able to do any further work in this course until you do.

Details
