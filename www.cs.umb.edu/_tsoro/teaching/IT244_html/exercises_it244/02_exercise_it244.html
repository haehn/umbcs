Logging in to users3

We'll be using the Windows machines in this room, or your laptops , to connect to the Linux machine users3 for all the class exercises in this course.

  • Install putty.exe if you have a Windows machine
    Download it from https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html.
    For more information go to https://www.cs.umb.edu/~tsoro/teaching/linux/remote_access_windows.html
  • Run the Terminal program if you have a Mac
    You will find this program in the Utilities directory inside the Applications directory.
    For more information go to https://www.cs.umb.edu/~tsoro/teaching/linux/remote_access_mac.html
  • Run the ssh client
    Use the values in the table below to connect to the virtual machine we'll be using in this class.

    In the value for "host" listed below, the first part of the string, "users3", specifies the machine, on the Unix network, that you will connect to.

    ssh configuration parameter value
    protocolssh
    ssh version2
    ssh port22
    hostusers3.cs.umb.edu
    user name your Unix username
    Authentication methodpassword

  • Enter your password
    A dialog box will appear on the screen into which you must type the password for your Unix account.
  • Confirm that you are connected
    You should see the following
    Welcome to Ubuntu 16.04.1 LTS (GNU/Linux 4.4.0-59-generic x86_64)
  • Learn More About the Machine to Which You Are Connected
    The uname command displays information about the machine you are using. 
    uname -a
    When used with the -a option, uname displays more information than just the name of the machine.

Your Shell

  • Determine Your Current Shell
    The easiest way to determine your current shell is to use the ps (process status) command. 
    ps
    The ps command shows all your running processes, including your shell, bash.

Explore Your Home Directory

  • Learn Your Current Location
    At every point in your Unix session, you are working inside some directory.

    When you log in, you will be in your home directory.

    You can create subdirectories inside your home directory, and customize your environment through special files located here.

    To determine your current location, you will use the pwd command, which stands for "print working directory". 
    pwd
    Notice that your home directory has the same name as your username, and is located inside /home.

    This is true for all users.
  • Look at the Contents of Your Home Directory
    The ls command (for "list"), when used with no arguments, will display the contents of your current directory. 
    ls
    You should see at least one directory - it244.

    You will use this directory for all your class assignments.
  • Look at Your "Invisible" files
    Normally ls will not show files whose first character is a . (dot).

    These are special files used to change features of your Unix environment.

    To see these files, you need to use ls with the -a option: 
    ls -a

Create directories for your homework and Class Exercises

  • Make sure you are in your home directory
    	cd
  • Go to your it244 directory
    	cd it244
  • Create an hw directory
    	mkdir hw
  • Create an ex directory
    	mkdir ex
  • Check that the directories were created
    	ls
    You should see two directories hw and ex
