diff

  • Go to the files directory for this course
    	cd  /home/tsoro/course_files/it244_files
  • Examine the contents of numbers1.txt and numbers2.txt 
        cat  numbers1.txt
    cat  numbers2.txt
  • Use diff to compare numbers1.txt and numbers2.txt 
        diff  numbers1.txt  numbers2.txt
    The output is hard to read, because it contains instructions for the Unix patch utility to convert the first file given at the command line to the second.



  • Use the -y option to diff to create readable output 
        diff  -y  numbers1.txt  numbers2.txt
    This output is much easier to read.

Pipes

  • Make sure you are in your home directory
        cd
  • Go to your ex6 directory
        cd it244/ex/ex6
  • Make sure you are in your ex6 directory
        pwd
  • Copy red_sox.txt to your ex6 directory
        cp  /home/tsoro/course_files/it244_files/red_sox.txt  .
    Make sure you include the dot, ., or you will get an error message.

  • Make sure the copy operation worked
        ls
  • Print the red_sox.txt
        cat  red_sox.txt


  • Find all Red Sox wins
        grep Win red_sox.txt
  • Find all Red Sox wins at home
        grep Win red_sox.txt | grep vs
    Here, the first instance of grep is the source, and the second is a filter.

  • Find all Red Sox wins at home, sorted in reverse order by date
        grep Win red_sox.txt | grep vs | sort -nr
    This is a pipeline, with three separate processes, one for each command.

finger

  • Use finger to see who else is connected to your current host 
        finger
  • Use finger on your own Unix username 
        finger YOUR_UNIX_USERNAME
    In place of YOUR_UNIX_USERNAME, you must put your Unix username.

  • Run finger on the first name "Glenn"
