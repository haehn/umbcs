Appending Output to a File

  • Use echo to send a string to a file.
        echo foo > lines.txt
  • Use cat to display the contents of lines.txt
        cat lines.txt
  • Write another string to lines.txt
        echo bar > lines.txt
  • Use cat to display the contents of lines.txt
        cat lines.txt
    Notice that the previous contents of lines.txt have vanished.

  • Send another line to lines.txt
        echo bletch >> lines.txt
  • Use cat to display the contents of lines.txt
        cat lines.txt
    Notice that your first line is still there.

  • Send still another line to the file
        echo bling >> lines.txt
  • Run cat on lines.txt
        cat lines.txt

/dev/null

  • Use echo to display a string in your terminal session window
        echo bonkers
  • Now perform the same operation, but send output to /dev/null
        echo bonkers > /dev/null
    Nothing appears in the terminal window since you have sent the string "bonkers" into oblivion by redirecting the output of echo to /dev/null.
